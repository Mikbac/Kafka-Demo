# Security

[https://kafka.apache.org/documentation/#security](https://kafka.apache.org/documentation/#security)

Kafka supports:

* SSL/TLS
* SASL

Possible options for the security protocol:

* PLAINTEXT
* SSL
* SASL_PLAINTEXT
* SASL_SSL
